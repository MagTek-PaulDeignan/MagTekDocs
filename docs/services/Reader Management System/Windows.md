# Windows

* Windows Application (PN 1000009xxx)
* Compatible with both USB and Bluetooth Low Energy (BLE) connected readers, as well as WLAN-enabled readers.
* Supports MagTek MagneSafe and DynaFlex/DynaProx readers.
